Projects

Braille Buddy

Braille Buddy was the 2019 Grand prize winner of the Tartan-hacks hackathon at Carnegie Mellon University. It converts English text into braille using Arduino motors. It works by converting each character into an integer code which corresponds to which dots need to be raised on the device. I.e.  'e' is equivalent to '15' which means the first and fifth dot get raised. 

Credible Collector (Creddy)

The Credible Collector was an entry in the 2020 SteelHacks hackathon at the University of Pittsburgh. The goal of the project was to create a graph of papers. Each paper would be connected to another paper which either it cited or which cited it. This would allow students to visualize the credibility of a paper they wish to use for research, where heavily cited papers would be assumed to have the most credibility.

RazzyBot

RazzyBot is a currently in progress personal project. The goal is to create an automated twitch stream utilizing a Raspberry Pi. The audience would play against the pi itself at various games. Some other features would be for it to start/stop the stream automatically, moderate chat automatically, and to scrape the chat in order to choose the audience input.
